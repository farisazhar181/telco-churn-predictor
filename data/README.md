# Telco Customer Churn Dataset

The dataset used in this project is the "Telco Customer Churn" dataset from Kaggle.

**Download Instructions:**
1. Visit the Kaggle dataset page: [Telco Customer Churn](https://www.kaggle.com/datasets/blastchar/telco-customer-churn)
2. Download the dataset as a CSV file.
3. Place the downloaded file (`WA_Fn-UseC_-Telco-Customer-Churn_2.csv`) in the `data/raw/` directory of this project.

**Note:** The raw data files (`*.csv`) are not tracked by version control (`.gitignore`).
